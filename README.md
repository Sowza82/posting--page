# Posting Page | Projeto de Certificação 2 

## 📘 Resumo do Projeto

Este projeto consiste em uma página web para criação de posts, simulando o funcionamento de redes sociais como Facebook ou LinkedIn. O usuário pode digitar um **título** e um **conteúdo**, visualizar o **preview** em tempo real e publicar o post, que será renderizado como um **card** abaixo do formulário. Além disso, os posts são enviados para uma API aberta (`https://jsonplaceholder.typicode.com/posts`) utilizando **fetch**.

O projeto foi desenvolvido com foco em **JavaScript**, **HTML semântico** e **CSS moderno**, garantindo interatividade, responsividade e uma boa experiência de usuário.

---

## 📘 Funcionalidades

- Criar post com **título** e **conteúdo**.
- Visualizar **preview** em tempo real.
- Publicar posts como **cards** dinâmicos.
- Envio do post para a API externa (`POST` via fetch).
- Limpeza do formulário após publicar.
- Interface responsiva e estilização moderna.

---

## 📘 Tecnologias Utilizadas

- HTML5 semântico
- CSS3 (Grid, Flexbox, sombras, hover effects)
- JavaScript (DOM, eventos, fetch API)
- API aberta: [JSONPlaceholder](https://jsonplaceholder.typicode.com/)
